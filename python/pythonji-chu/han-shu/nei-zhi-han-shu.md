dir(__builtins__)

>>> print(max(a))

### 高级内置函数

```py
enumerate 返回一个可以枚举的对象
exec      执行字或complie方法编译过的字符串
map       对于参数iterable中的每个元素都应用fuction函数，并将结果作为列表返回
eval      1.取出字符串中的内容
          2.将字符串str当成有效的表达式来求指并返回计算结果
filter    过滤器
zip       将对象逐一配对

--------------------------------------------------------------------
返回一个枚举对象（枚举就是把一个列表的索引和元素返回出来）也可以转换为字典返回k和值
>>> li = ['a','b','c','d']
>>> print(list(enumerate(li)))

eval可以识别表达式
>>> a = '1+2+3'
>>> print(eval(a))
exec可以识别赋值语句
>>> exec('a=1')

filter
def func(x):                #过滤函数
    return x > 10
li = [2,5,8,11,22,24]       #过滤条件
y = list(filter(func,li))   #filter来进行过滤
print(y)

map
def fun2(num):
    return num * 10
li = [1,2,3,4,5]
f = list(map(fun2,li))
print(f)

zip配对
t1 = [1,2,3]
t2 =['a','b','c']
tt = list(zip(t1,t2))
print(tt)

在列表中没有的不配对 可以转换为任意数据类型
t2 =['a','b','c']
t3 = ['aaa','bbb','ccc','ddd']
t4 = list(zip(t2,t3))
print(t4)
```

常见内置函数提供了一些处理的数据的方法，可以帮助我们提高开发速度
