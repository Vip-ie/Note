### 列表

**列表特点：有序、索引下标**

在python中有东西既可以放字符串又可以放数值吗？

### 列表的定义

列表由变量复制中括号括起来，元素与元素之间用逗号隔开，列表是有顺序的、它的索引值就是列表的顺序。

```py
list_a = [1,2,3,'a','b','c']
```

### 列表转换数据类型方法

```py
>>> l = str(list_a)
>>> l
"[1, 2, 3, 'a', 'b', 'c']"
----------------------------------------------
list_a = [1,2,3,'a','b','c']
>>> ls = tuple(list_a)
>>> ls
(1, 2, 3, 'a', 'b', 'c')
```

### 索引取值

列表为什么能索引取值，字符串，列表、元组都是有顺序的，所以可以索引取值。

### 切片取值

```py
>>>lists = [1,2,3,4,5]
>>>lists[2:]           从当前索引位置取到最后
3
>>> lists[1:3]         索引取值索引位1到3的元素
[2, 3]
>>> lists[1:5:2]       最后一个元素叫步长
[2, 4]
```

### **列表操作方法**

```py
增:    append(增加一个元素)    insert(插入一个元素)    extend(插入多个元素)
删:    clear(清空列表所有元素)    pop(删除一个指定下标元素)    remove(删除一个指定元素)    del
改:
查:    index(查看一个指定元素)    count(查看某个元素出现次数)
其他方法:    copy(浅复制)copy.deepcopy(深复制)    reverse(降序)    sort(升序)


------------------------------------------------------------

创建一个列表
    例1:li = [1,2,3,'a','b']
------------------------------------------------------------

为列表增加一个元素
    例2:li.append('c')
------------------------------------------------------------

为列表指定元素位置插入元素(插入位置有下标指定)
    例3:li.insert(1,'bb')
------------------------------------------------------------

为列表扩展多个元素
    例4:li.exted(['abc','uc'])
------------------------------------------------------------

删除列表最有一个元素(如果指定元素有下标指定)
    例5:li.pop(0)
------------------------------------------------------------

删除列表指定元素(指定元素名称)
    例6:li.remove('uc')
------------------------------------------------------------

清空一个列表所有元素
    例7:li.clear()
------------------------------------------------------------

查看一个元素的索引值(指定元素名称查看索引值)
    例8:li.index(1) 或li.index('a')
------------------------------------------------------------

查看一个元素在列表中出现的次数
    例9:li.count('a') 或li.count(2)
------------------------------------------------------------

列表排序sort(升序)reverse（降序）【同类型可以排序，不是同类型可以转换为同类型排序】
    例10:li.sort() 或li.reverse()
------------------------------------------------------------

浅复制获得一个新列表
    例11:li1 = li.copy()
------------------------------------------------------------

深复制获得一个新列表
    例12: import copy  li2=copy.deepcopy(li)
```

### 深浅复制的应用

深浅复制只有在列表嵌套列表的情况下讨论

如果想保留修改之前的数据，就可以使用列表的复制，但要注意列表嵌套情况下的问题

**浅复制**

```py
浅复制只在列表嵌套列表是使用
浅拷贝嵌套列表随着列表内容改变
>>> li = [1,2,3]
>>> li2 =['a',li]
-------------------------------------
>>> li3 = li2.copy() 
>>> id(li2[0])
4437568792
>>> id(li3[0])
4438277624
-------------------------------------
>>> li2[0]='u'
>>> li2
['u', ['f', 2, 3]]
>>> li3
['a', ['f', 2, 3]]
-------------------------------------
>>> li[0]='cc'
>>> li2
['u', ['cc', 2, 3]]
>>> id(li2[1][0])
4439333608          
>>> li3
['a', ['cc', 2, 3]]
>>> id(li3[1][0])
4439333608
```

**深复制**

```py
深复制可以拷贝一个列表到新的内存地址
>>> li = [1,2,3]
>>> li2 =['a',li]
>>> import copy
>>> li4 = copy.deepcopy(li2)
>>> li4
['a', [1, 2, 3]]
-------------------------------------
>>> li[0]=99
>>> li2
['a', [99, 2, 3]]
>>> li4
['a', [1, 2, 3]]
```
