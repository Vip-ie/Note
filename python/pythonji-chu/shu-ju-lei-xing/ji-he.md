### 集合

**集合特性**

**唯一性：**集合中的元素具有唯一性，不存在两个相同的元素

**无序性：**集合中的元素是无序的，所以不存在索引

**可变性：**集合是可变对象

### 两种集合定义方法

```py
a = set（）定义个一空集合
a = {1,2,3,}
```

### 三种运算

**交集：两个集合中都有的元素【&】**

**并集：两个集合中所有的元素【\|】**

**差集：剩下的元素 【-】**

```py
>>> a ={1,2,3,4,}
>>> b ={2,3,5,6}
>>> c = {1,2,3,4,5,6,7,8,9}
----------------------------------------------

交集
>>> a & b
{2, 3}
----------------------------------------------

并集
>>> a | b
{1, 2, 3, 4, 5, 6}
----------------------------------------------

差集
>>> c - a
{5, 6, 7, 8, 9}
```

```py

删:
    pop
    remove
查:
    isdisjoint  有交集返回False
    issubset    判断是否包含于
    issuperset  判断是否包含
----------------------------------------------

增
>>> a
{1, 2, 3, 4}
>>> id(a)
4441525608
>>> a.add(666)
>>> a
{1, 2, 3, 4, 666}
>>> id(a)
4441525608

>>> a.update('hello')
>>> a
{1, 2, 3, 4, 'l', 'o', 'h', 'e', 666}
----------------------------------------------

删
>>> a.pop()
1
>>> a
{2, 3, 4, 'l', 'o', 'h', 'e', 666}

>>> a.remove(666)
>>> a
{2, 3, 4, 'l', 'o', 'h', 'e'}
----------------------------------------------

查
>>> a.isdisjoint(b)
False
>>> a.issubset(c)
True
>>> a.issuperset(b)
False
```
