**条件语句**

```py
条件满足，执行if语句
条件不满足，执行else语句
----------------------------------------------
if 判断语句1:
    执行语句1
else:
    执行语句
----------------------------------------------

条件满足，执行if语句

leva = '男'
if leva == '男':
    print('leva是男生')
else:
    print("leva是女生")
----------------------------------------------

条件不满足，执行else语句

leva = '女'
if leva == '男':
    print('leva是男生')
else:
    print("leva是女生")
```

**条件判断**

```py
if 判断语句1:
    执行语句1

elif 判断语句2
    执行语句2

else:
    执行语句
----------------------------------------------

a = input("请输入:")        #请输入信息
if a.isdigit():            #判断输入的是不是数字
    a = int(a)             #判断a输入的是不是整数  
    if a >= 90:            #判断a如果大于等于90
        print("A")         #输出A
    elif a >= 80:          #判断a如果大于等于80
        print("B")         #输出B
    elif a >= 60:          #判断a如果大于等于60
        print("C")         #输出C
    else:
        print("差")        #输出差
else:
    print("输入错误")       #输出 输入错误
```

### 三目运算

**可以把简单的条件判断写成一行**

**三目运算方法**

值1 if 判断语句 else 值2  **（**当值1满足条件执行if判断，条件不满足执行else条件）

```py
有这样一个判断，如果值大于5返回True，否则返回False，代码怎么写？
-----------------------------------------------------
传统方法

a = 10
if a > 5:
    print('True')
else:
    print('False')
-----------------------------------------------------

三目运算方法
a = 10
print(True) if a  > 5 else print(False)

b = True a if > 5 else b = False
print(b)
```
