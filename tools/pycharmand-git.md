#### **Pycharm配置git**

**OSX配置git用户名和邮件地址**

打开终端输入一下命令：

```
查看git用户名和邮箱，如果没有配置git用户名
git config --global user.name
git config --global user.email
配置用户名和邮箱
git config --global user.name"leva"
git config --global user.email ilovezlk@126.com
```
