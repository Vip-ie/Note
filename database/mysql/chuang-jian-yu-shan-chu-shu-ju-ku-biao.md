#### MySQL 创建表

##### 创建表 语法

##### create table \[if not exists\] table\_name\(

##### column\_name data\_type,

##### \);

```
mysql> create table if not exists student(
    -> id int,
    -> name varchar(10)
    -> );
Query OK, 0 rows affected (0.06 sec)
```
